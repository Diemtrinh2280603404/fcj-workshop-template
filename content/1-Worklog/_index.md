This worklog documents the entire internship journey at **Amazon Web Services Vietnam** over **12 weeks** (from April to July 2026). Throughout the program, I progressed from learning cloud computing fundamentals to designing and deploying a complete real-world system on AWS.

The 12-week internship is divided into two main phases:

**Phase 1 – AWS Foundations (Weeks 1–7):** Studied core AWS services including EC2, S3, VPC, RDS, IAM, CloudFront, Route 53, SageMaker, Lambda, EventBridge and security services. Reviewed and prepared for the AWS certification exam.

**Phase 2 – FlashLearn Project (Weeks 8–12):** Built the FlashLearn English learning application on AWS with full features: Flashcard, Quiz, Battle, community interaction, ranking system, and project handover.
