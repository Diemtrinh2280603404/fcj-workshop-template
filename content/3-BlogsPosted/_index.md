Below are the blog posts I published on [AWS Study Group](https://www.facebook.com/groups/awsstudygroupfcj) during my internship at Amazon Web Services Vietnam.

### [Blog 1 - OPTIMIZING SECURITY WITH SESSION POLICIES IN AMAZON EKS POD IDENTITY](3.1-Blog1/)
This blog introduces the Session Policies feature in Amazon EKS Pod Identity – allowing IAM permissions to be scoped flexibly and precisely for each Pod without creating additional IAM Roles. This is an important advancement that helps apply the Least Privilege principle more effectively in large-scale Kubernetes environments.

### [Blog 2 - EVENT-DRIVEN ARCHITECTURE ON AWS: EVENTBRIDGE, SNS AND SQS IN SERVERLESS SYSTEMS](3.2-Blog2/)
This blog presents the Event-Driven Architecture model on AWS, where components communicate via Events instead of direct REST API calls. The article analyzes the role of Amazon EventBridge, Amazon SNS and Amazon SQS in reducing coupling, increasing scalability and ensuring asynchronous processing in Serverless systems.

### [Blog 3 - BUILDING A DATA LAKE WITH AMAZON S3 AND AMAZON ATHENA](3.3-Blog3/)
This blog introduces the Data Lake architecture on AWS using Amazon S3 to store raw data, AWS Glue to manage Metadata, and Amazon Athena to query data with SQL without managing infrastructure. This solution helps reduce storage costs, increase scalability and provides a foundation for data analytics and Machine Learning applications.
